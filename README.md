# AK Barrierefreiheit in der Bildung (IAAP-DACH)

Der Arbeitskreis "Barrierefreiheit in der Bildung" befasst sich mit den folgenden Themenblöcken:
1. Curriculum
2. Zertifizierung
3. Gemeinsame Projekte
4. Sensibilisierung & Awareness
5. Best Practices

Wir treffen uns jeden 2. Monat. Task Forces machen die eigentliche Arbeit zwischen den Meetings.
